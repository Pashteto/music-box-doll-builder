## Implementation Steps

### 1. Scaffold the Next.js project

Run the create-next-app CLI with the required options:

```bash
npx create-next-app@latest . \
  --app --typescript --tailwind --eslint
```

Confirm that React 19 and Next.js 15 are listed in `package.json`. If the CLI installs an older version, manually upgrade both packages to the latest stable releases.

### 2. Enable strict TypeScript

Open `tsconfig.json` and add the following under `compilerOptions`:

- `"strict": true`
- `"noUncheckedIndexedAccess": true`

### 3. Configure path aliases

In `tsconfig.json`, ensure the path alias is set:

```json
"paths": {
  "@/*": ["./src/*"]
}
```

### 4. Create the folder structure

Create all planned directories. Each empty directory should contain a `.gitkeep` file so Git tracks it.

**Top-level directories inside `src/`:**

- `src/app/` (created by Next.js)
- `src/components/`
- `src/modules/`
- `src/lib/`
- `src/store/`
- `src/hooks/`

**Module subdirectories inside `src/modules/`:**

- `editor/`
- `scene-composer/`
- `catalog/`
- `paywall/`
- `render/`
- `share/`
- `analytics/`
- `storage/`

**Public assets:**

- `public/assets/`

### 5. Add .nvmrc

Create `.nvmrc` at the project root with contents: `20`

This pins the project to Node 20 LTS.

### 6. Verify dependency versions

Open `package.json` and confirm:

- `react` and `react-dom` are version 19.x
- `next` is version 15.x

If they are not, upgrade manually with `npm install react@19 react-dom@19 next@15`.

### 7. Verify path alias resolution

Create a trivial utility file at `src/lib/hello.ts` that exports a string constant. Import it in `src/app/page.tsx` using `@/lib/hello` and render the value. Confirm the dev server and build resolve the alias without errors.

## Completion Criteria

- [ ] `npm run dev` starts without errors
- [ ] `npm run build` succeeds with zero TypeScript errors
- [ ] Visiting `localhost:3000` shows the default Next.js page (or the modified test page)
- [ ] `tsconfig.json` has `"strict": true` and `"noUncheckedIndexedAccess": true`
- [ ] All planned directories exist (even if empty with `.gitkeep`)
- [ ] Path alias `@/lib/hello` resolves correctly in `src/app/page.tsx` during build
- [ ] `.nvmrc` contains `20`
- [ ] `package.json` lists React 19.x and Next.js 15.x
