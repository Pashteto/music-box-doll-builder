## Implementation Steps

### 1. Create the share module

Create `src/modules/share/shareVideo.ts`.

### 2. Implement the main share function

Export a function with the following signature:

```ts
shareVideo(blob: Blob, filename: string):
  Promise<'shared' | 'downloaded' | 'cancelled'>
```

### 3. Create a File object from the Blob

```ts
const file = new File(
  [blob], filename, { type: 'video/mp4' }
);
```

### 4. Check sharing support

Call `navigator.canShare({ files: [file] })` to verify that the browser supports file sharing via the Web Share API.

### 5. Invoke the native share sheet

If supported, call `navigator.share({ files: [file], title: 'My Music Box Doll' })`. Handle the returned promise:

- Resolve on success (return `'shared'`).
- Catch `AbortError` when the user cancels (return `'cancelled'`).
- Catch other errors and log them.

The native share sheet should display Instagram Reels/Stories, WhatsApp, Telegram, and other installed apps as destinations.

**Important:** Instagram file sharing is unreliable on some devices — document this as a known limitation.

### 6. Fall through to download on unsupported browsers

If `navigator.share` with files is not supported, fall through to the download fallback (E11-T2).
