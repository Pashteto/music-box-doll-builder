## Implementation Steps

### 1. Create the SceneComposer component

Create `src/modules/scene-composer/SceneComposer.tsx` as a `'use client'` component. This is the main scene composer screen where the user decorates the music box scene after global adjustment.

### 2. Set editor mode in Zustand

On mount, set `editorMode` to `'scene'` in the Zustand store. This signals that the user is in the scene decoration phase.

### 3. Render the assembled doll in DollScene

Render `<DollScene>` showing the fully assembled doll with all placed assets and their final transforms (including any global adjustments from E7).

### 4. Display background catalog as horizontal scrollable strip

Load backgrounds from the catalog manifest via `getBackgrounds()` (or equivalent catalog accessor). Render them as a horizontal scrollable strip of thumbnail cards (same pattern used in the slot asset catalog). The first option should be "Empty/None" — representing no background (default neutral scene background).

### 5. Handle background selection

On tapping a background thumbnail, update `sceneBackground` in the Zustand store with the selected asset reference. If "Empty" is selected, set `sceneBackground` to `null`.

### 6. Render selected background in DollScene

Inside DollScene, read `sceneBackground` from the store. If set, load and render the background asset behind the doll. Depending on the asset type this could be a 3D skybox mesh, a textured plane positioned behind the doll, or an environment map. The doll should remain correctly lit regardless of the background.

### 7. Enable real-time preview

Background changes should be reflected immediately in the 3D scene. Use Suspense boundaries for the GLB/texture loading to show a brief loading indicator without blocking the UI.

### 8. Trigger autosave on selection change

After the user selects or changes a background, trigger the autosave mechanism (debounced) to persist the choice to IndexedDB.

## Completion Criteria

- [ ] Background catalog shows available backgrounds plus an "Empty/None" option
- [ ] Selecting a background updates the 3D scene in real-time
- [ ] Selecting "Empty" removes the background (returns to neutral default)
- [ ] Doll remains visible and correctly lit with any background applied
- [ ] Catalog is horizontally scrollable with thumbnail previews
- [ ] Autosave triggers on selection change
- [ ] `editorMode` is set to `'scene'` while this screen is active
