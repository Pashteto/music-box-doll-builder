## Implementation Steps

### 1. Install the Stripe Go SDK

```bash
go get github.com/stripe/stripe-go/v76
```

### 2. Create the handler

Create `internal/checkout/handler.go`.

### 3. Define the endpoint

`POST /api/v1/checkout/session`

Request body: `{ sessionId, productId, successUrl, cancelUrl }`

### 4. Validate the request

- `sessionId` is required.
- `productId` must match a known product.
- `successUrl` and `cancelUrl` must be valid URLs.

Return 400 with a descriptive error message for invalid input.

### 5. Create a Stripe Checkout Session

Configure the Stripe session:

- Mode: `payment` (one-time).
- Line items: look up the Stripe Price ID for the given `productId`.
- Success URL and Cancel URL from the request body.
- Client reference ID: the `sessionId` (links the payment back to the anonymous session).
- Metadata: `sessionId`, `productId`.

### 6. Return the checkout URL

Return `{ checkoutUrl: session.URL }` with HTTP 200.

### 7. Configure the Stripe secret key

Read the Stripe secret key from an environment variable (`STRIPE_SECRET_KEY`). Never log or expose this value.

### 8. Add rate limiting middleware

Apply rate limiting to this endpoint to prevent abuse. Return HTTP 429 when the limit is exceeded.
